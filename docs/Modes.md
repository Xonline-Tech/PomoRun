# 预设模式配置（MVP）

本文件把“表格描述”转成可执行的模式配置与脚本化分段（segments）。

## 字段约定

- 建议时间：MVP 默认固定总时长（可选允许在范围内调整）
- 步频（步/分钟）：对应节拍器 BPM（beats per minute），MVP 用固定值
- 语音提醒：用规则表达（每 5 分钟/剩余 5 分钟/间歇切换前 10 秒）

## 模式一览（人类可读）

### 1) 轻松慢跑

- 建议时间：30 分钟
- 速度感受：能正常聊天，不喘
- 步频：160
- 适合人群：新手、减脂、养生、恢复跑
- 语音提醒：每五分钟提醒 + 鼓励
- 注意事项：步幅小，不追求快

segments：

- steady 30:00 @ 160

### 2) 间歇跑

- 建议时间：10 分钟
- 速度感受：冲刺时很累，间歇能缓过来
- 步频：快跑 200；恢复 130
- 适合人群：想高效减脂、提升体能
- 语音提醒：快跑 30s + 慢跑 1.5min；每次切换前十秒提醒 + 鼓励
- 注意事项：热身必须足，不天天练

segments（5 组，共 10 分钟）：

- repeat 5x:
  - fast 0:30 @ 200
  - recovery 1:30 @ 130

### 3) 稳态长跑

- 建议时间：60 分钟
- 速度感受：全程舒服稳定，微喘
- 步频：170
- 适合人群：练耐力、5km/10km 进阶
- 语音提醒：每五分钟提醒 + 鼓励
- 注意事项：补水，配速别越跑越快

segments：

- steady 60:00 @ 170

### 4) 节奏跑

- 建议时间：15 分钟
- 速度感受：有点累，但能坚持到底
- 步频：185
- 适合人群：想提速、突破瓶颈
- 语音提醒：剩余 5 分钟、1 分钟提醒
- 注意事项：不要当成冲刺跑

segments：

- steady 15:00 @ 185

### 5) 超慢跑

- 建议时间：40 分钟
- 速度感受：几乎不喘，像“颠着走”
- 步频：150
- 适合人群：大体重、怕伤膝、长期减脂
- 语音提醒：每五分钟提醒 + 鼓励
- 注意事项：步幅极小，落地轻

segments：

- steady 40:00 @ 150

## 可直接映射为代码常量的建议（示例结构）

下面是“形状示例”，用于指导你在 Dart 中落常量（字段名可按实现调整）：

```txt
ModeConfig {
  id: "easy_jog",
  name: "轻松慢跑",
  defaultDurationSec: 30 * 60,
  bpm: 160,
  segments: [
    Segment(type: steady, durationSec: 30*60, bpm: 160)
  ],
  prompts: [
    PromptRule.everyMinutes(5, template: "坚持住，保持轻松节奏")
  ],
  notes: ["步幅小，不追求快"],
}
```
